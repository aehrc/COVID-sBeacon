This heatmap is generated by using a sliding 14-day window approach. The value for each window is calculated as the ratio of new mutant samples against new wild type samples. A minimum of 10 samples must be present for a window to be valid. The heatmap values can then be toggled between a log2 calculation of the ratio or one which is capped at 2 (ie when the mutant strain appears twice as frequently as the wild type).
